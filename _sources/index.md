# PF-3311 Ciencia de datos geoespaciales

Este curso trata sobre el manejo, visualización y análisis de datos geoespaciales mediante herramientas computacionales, incluyendo sistemas de información geográfica y bibliotecas especializadas de lenguajes de programación, como Python y R. Se introducen conceptos básicos de datos geográficos y se presenta una visión general de cómo metodologías y técnicas de ciencia de datos y aprendizaje automatizado pueden ser aplicadas al componente geoespacial de diversos problemas.

El curso forma parte del [Programa de Posgrado en Computación e Informática](http://www.pci.ucr.ac.cr/) de la [Universidad de Costa Rica](https://www.ucr.ac.cr/).
